# Page Map

## `index.html`

Future route concept:

- `/[lang]/help-center/`

Purpose:

- Help center landing page.

Main sections:

- SiteHeader;
- HelpHero;
- HelpSearchBar;
- CategoryGrid;
- SiteFooter.

Future additions:

- popular articles;
- recommended topics;
- contact support band;
- announcement / maintenance notice if required.

## `category.html`

Future route concept:

- `/[lang]/help-center/[category]/`

Purpose:

- Show a category title and article list.

Main sections:

- Breadcrumb;
- CategoryHeader;
- ArticleList;
- ArticleCard.

Future additions:

- subcategory tabs;
- left category sidebar;
- pagination or load more;
- contact support card.

## `article.html`

Future route concept:

- `/[lang]/help-center/[category]/[slug]/`

Purpose:

- Show help article content.

Main sections:

- Breadcrumb;
- ArticleHeader;
- ArticleBody;
- ArticleToc;
- HelpfulFeedback.

Future additions:

- related articles;
- article updated date;
- share / copy link action;
- print-friendly layout if required.

## `search.html`

Future route concept:

- `/[lang]/help-center/search/?q=...`

Purpose:

- Show search input, results, and empty state.

Main sections:

- SearchHeader;
- HelpSearchBar;
- SearchResultList;
- EmptyState.

Future additions:

- result count;
- category filters;
- highlighted query snippets;
- loading skeleton;
- typo suggestions.
